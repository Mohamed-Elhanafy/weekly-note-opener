# Weekly Note Opener

An Obsidian plugin that automatically opens the current week's note, creating it if it doesn't exist.

## Features

-   **One-Click Access**: Open your weekly note from the ribbon icon or using the command palette ("Open Weekly Note").
-   **Auto-Creation**: If the note for the current week doesn't exist, it is created automatically.
-   **Configurable**: Set your preferred folder path and date format (e.g., `YYYY-[W]ww`) in the settings.

## Installation

### From Community Plugins
1.  Open Obsidian Settings > Community Plugins.
2.  Turn off "Safe mode".
3.  Click "Browse" and search for "Weekly Note Opener".
4.  Install and Enable.

### Manually
1.  Download the latest release from the [Releases](https://github.com/Mohamed-Elhanafy/weekly-note-opener/releases) page.
2.  Extract the `main.js`, `manifest.json`, and `styles.css` files into your vault's plugin folder: `.obsidian/plugins/weekly-note-opener/`.
3.  Reload Obsidian and enable the plugin in Settings.

## Settings

-   **Weekly Notes Folder**: The directory where your weekly notes are saved (default: `2 - areas/journals/weekly`).
-   **Date Format**: The file naming format using Moment.js tokens (default: `YYYY-[W]ww`).
